التراجع عن إعادة النشر (Undo Repost)

سيتم إلغاء إعادة النشر من التغريدة.

API سريع

Endpoint: POST /api/undo-repost (Bearer Token)
curl -X POST http://localhost:5789/api/undo-repost \
  -H "Authorization: Bearer your-secure-token-here" \
  -H "Content-Type: application/json" \
  -d '{"cookie_label":"acc1","tweet_url":"https://x.com/.../status/...","headless":true}'
